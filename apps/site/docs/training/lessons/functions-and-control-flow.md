# Functions and Control Flow

This lesson introduces how programs make decisions, repeat work, and organize reusable logic.

## Lesson goal

- understand `if`, `else`, and looping patterns
- know why functions help keep code reusable and readable
- build confidence with basic logic structures

## What to teach

- conditional decisions
- loops for repeated work
- function inputs and outputs
- keeping one function focused on one job

## Real app examples

- validate whether a password is strong enough
- loop through cart items to calculate a total
- create a function that formats a user’s display name
- show different messages based on login state

## Practice ideas

- build a simple discount calculator
- create a function that checks whether a form field is empty
- use a loop to filter items based on a condition

## Teaching outcome

- learners start writing reusable logic instead of repeating code blindly
