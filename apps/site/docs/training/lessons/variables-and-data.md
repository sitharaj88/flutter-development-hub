# Variables and Data Types

Variables are how programs remember things. This lesson helps learners understand what kinds of data exist and how to work with them safely.

## Lesson goal

- understand what a variable is
- identify common data types
- choose the right kind of data for a simple task

## What to teach

- text, numbers, booleans, lists, and maps
- mutable and fixed values
- naming variables clearly
- why wrong data types cause errors and confusion

## Real app examples

- username and email as text
- age, total, and rating as numbers
- isLoggedIn as a boolean
- product list as a collection

## Practice ideas

- create variables for a sign-up form
- store product information using multiple data types
- explain why a number should not be stored as plain text unless necessary

## Teaching outcome

- learners gain confidence storing and reasoning about application data
