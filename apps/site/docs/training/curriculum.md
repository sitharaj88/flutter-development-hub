# Curriculum

The curriculum is designed as a complete learning ladder from programming to app design, app development, and final delivery quality.

## Module 1: Programming Fundamentals

- logic building, variables, conditions, loops, and functions
- breaking problems into steps and writing small solutions
- basic debugging mindset and code-reading confidence

## Module 2: Dart for App Development

- Dart syntax and type system
- collections, functions, classes, and null safety
- async thinking and practical data handling

## Module 3: Flutter Core

- widgets, layout systems, navigation, and reusable UI
- stateful interactions and form flows
- building complete screens instead of isolated fragments

## Module 4: App Design and UX

- flow planning, layout clarity, spacing, and component consistency
- responsive design and better presentation of app states
- turning rough ideas into cleaner user experiences

## Module 5: State and Architecture

- structuring medium-sized apps sensibly
- choosing state management approaches
- separating UI, logic, and data responsibilities

## Module 6: Backend and Data

- APIs, JSON, authentication, persistence, and cache thinking
- network states and resilient UI behavior
- organizing data access for readability and reuse

## Module 7: Testing and Release

- testing basics, debugging, performance awareness, and build preparation
- environment thinking and release-readiness
- improving software quality beyond just "works on my machine"

## Module 8: Capstone

- plan and build a complete mini application
- justify design and technical decisions
- present the result as a learning artifact, portfolio project, or internal reference

## Optional advanced extensions

- advanced state management patterns
- modular architecture for larger teams
- analytics, notifications, and more complex integrations
- code review workflow and engineering standards for team environments
