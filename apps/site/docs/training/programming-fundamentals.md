# Programming Fundamentals

Before Flutter becomes comfortable, learners need to feel comfortable with programming itself. This section helps absolute beginners and weak fundamentals learners build the base needed for app development.

## What this module should feel like

- beginner-friendly without being shallow
- practical enough to feel connected to real apps
- structured enough that learners can build confidence step by step
- honest about mistakes and confusion, because that is where learning usually happens

## Topics covered

- what programs do and how instructions are executed
- variables and data types
- conditions and decision-making
- loops and repetition
- functions and code reuse
- arrays, lists, and maps
- problem-solving by breaking a task into smaller steps

## Practical learning goals

- read simple code without panic
- write short programs that solve a defined task
- understand why structure matters before building large apps
- become comfortable with debugging basic errors

## Recommended lesson progression

1. understand how problems become steps
2. learn how values are stored and compared
3. practice decisions with `if` and `else`
4. repeat work with loops
5. move repeated logic into functions
6. model slightly richer data with lists and maps

## Teaching artifacts to include

- whiteboard or plain-language logic breakdowns
- short code snippets with explanation
- mini exercises after every concept
- recap questions to check understanding
- small assignments that combine multiple basics together

## Why this matters for Flutter

- app development becomes easier when logic is not confusing
- state and UI flows make more sense when the learner understands conditions and functions
- later architecture lessons become much stronger when the base is solid

## Exercises that fit this module

- calculator logic
- login validation rules
- list filtering and search logic
- simple menu-driven console-style tasks

## Strong outcome for this stage

- learners should be ready to start Dart with less fear and more reasoning ability
