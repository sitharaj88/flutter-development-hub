# Full Learning Path

This training is designed as a real learning journey, not a set of disconnected topics. A learner should be able to begin with programming fundamentals and move step by step toward designing, building, testing, and presenting a complete Flutter app.

## Stage 1: Programming mindset

- understand how programs think through input, output, conditions, repetition, and structure
- move from theory into writing small logic-based solutions
- build confidence reading and explaining code

## Stage 2: Dart language fundamentals

- learn the language used by Flutter in a practical way
- understand variables, functions, collections, classes, null safety, and async basics
- practice solving small programming problems before moving into UI

## Stage 3: Flutter foundations

- learn how Flutter builds screens using widgets
- understand layout, navigation, composition, and reusable UI pieces
- build simple but complete app flows

## Stage 4: App design and user experience

- translate ideas into usable screens and flow structure
- think about spacing, readability, hierarchy, and state presentation
- create interfaces that feel intentional instead of random

## Stage 5: Data, architecture, and delivery

- connect apps to APIs and persistence layers
- structure projects for growth
- test, debug, optimize, and prepare apps for release

## Stage 6: Capstone and presentation

- plan and build a complete mini application
- explain technical and design decisions clearly
- use the project as proof of skill, learning, and delivery readiness

## Detailed lesson path

- start with programming logic, variables, and functions
- move into Dart collections, classes, and async basics
- build UI with widgets, layouts, navigation, and forms
- add app design, responsiveness, and better UX thinking
- learn state, API integration, testing, and capstone presentation

## Tutorial library path

- use the lesson pages for concise teaching blocks
- use the tutorial library for broader in-depth concept explanations
- combine both with projects, assignments, and live demonstrations
