# Programming Foundations Overview

Programming Foundations is the first serious layer of the training. It is where learners stop seeing software as intimidating and start understanding how logic, structure, and step-by-step thinking work.

## What this track is meant to achieve

- build comfort with problem-solving before app complexity appears
- create confidence with variables, conditions, loops, and functions
- prepare learners for Dart syntax and Flutter application logic
- reduce the fear that comes from trying to jump directly into app development

## What learners should know by the end

- how to break a problem into smaller tasks
- how to store and use values safely
- how to make decisions with conditions
- how to repeat work with loops
- how to organize logic into functions

## Recommended teaching order

1. computational thinking
2. variables and data
3. conditions and loops
4. functions and modularity

## Good teaching style for this track

- explain every concept with a plain-language example first
- connect logic to real app situations such as login, cart totals, and search
- keep practice small and frequent instead of using one big exercise too early
- encourage learners to explain their reasoning aloud, not just type code

## Why this stage matters so much

When this stage is weak:

- Dart feels harder than it should
- Flutter code feels intimidating
- learners depend too much on copying

When this stage is strong:

- later topics become easier
- learners make better sense of syntax
- confidence builds earlier

## Bridge to the next stage

Once learners are comfortable here, Dart becomes easier to learn because the mental model for programming is already in place.
