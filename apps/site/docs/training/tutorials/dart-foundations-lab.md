# Dart Foundations Lab

This lab page supports the first half of the Dart track with structured practice.

## Lab 1: Profile data setup

Goal:
- practice `var`, `final`, `const`, and simple types

Task:
- create variables for app title, user name, user age, login status, and session timestamp
- explain which values should stay fixed and which should change

## Lab 2: Checkout logic

Goal:
- practice operators and expressions

Task:
- calculate a discounted total
- decide whether checkout is allowed
- display a message if the cart is empty

## Lab 3: Validation helpers

Goal:
- practice functions and parameters

Task:
- write helper functions for email presence, password length, and display label formatting
- refactor repeated validation into small reusable units

## Lab 4: Safe profile modeling

Goal:
- practice null safety and structured data

Task:
- create a `Profile` class with required and optional fields
- return safe fallback values when optional data is missing

## Lab 5: Collections challenge

Goal:
- practice lists, sets, maps, and class models

Task:
- store product names in a list
- store selected categories in a set
- store app settings in a map
- model one app entity as a class

## What a good result looks like

- code is readable
- variable intent is clear
- functions are focused
- data is structured sensibly
- learners can explain why they chose each approach
