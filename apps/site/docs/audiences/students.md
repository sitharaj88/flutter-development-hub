# For Students

Students don't need more random tutorials. They need a **clear path, real practice, and enough support** to move from confusion to delivery confidence.

## What this path solves

| Problem | How this path addresses it |
|---------|---------------------------|
| Passive tutorial watching | Project-led learning with assignments after each module |
| Weak language foundation | Dedicated Dart module including Dart 3 features (records, patterns, sealed classes) |
| Can't build real apps | Capstone project with guided architecture and code review |
| No structure or progression | 8-module curriculum from fundamentals to deployment |
| Outdated content | Modern stack: Material 3, GoRouter, Riverpod, Impeller |

## Student path structure

1. **Programming foundations** — Logic, data structures, and problem solving
2. **Dart in depth** — Types, null safety, async, Dart 3 patterns and sealed classes
3. **Flutter core** — Widgets, layouts, Material 3 theming, GoRouter navigation
4. **Design & UX** — Responsive design, animations, accessibility
5. **State & architecture** — Riverpod, feature-based structure, clean architecture
6. **Backend & data** — REST APIs with dio, code generation with freezed, persistence
7. **Testing & release** — Unit tests, widget tests, CI/CD, multi-platform deployment
8. **Capstone** — Portfolio-quality app demonstrating full-stack Flutter skills

## What students walk away with

- **Production-quality Flutter skills** covering 6 target platforms
- At least one **portfolio-ready capstone project** with clean architecture
- Confidence in **technical interviews** — ability to explain trade-offs and design decisions
- Strong habits around **code quality**, testing, and structured development
- Understanding of the **modern Flutter ecosystem** (Dart 3, Material 3, CI/CD)

## Best-fit scenarios

- Final-year students building employable mobile skills
- Self-driven learners who want more than YouTube-style exposure
- Junior developers who know some Flutter but need deeper understanding
- Learners preparing for **cross-platform developer roles**
